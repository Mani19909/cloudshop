
# CloudShop - Production Grade E-Commerce Platform on AWS

## Project Overview
CloudShop is a production-grade DevOps project that demonstrates a complete CI/CD pipeline using AWS, Terraform, Jenkins, Docker, Kubernetes, Prometheus, and Grafana.

## Architecture

Developer → GitHub → Jenkins → Docker → ECR → EKS → Prometheus → Grafana
